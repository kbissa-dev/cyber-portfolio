# Labs

Hands-on labs and notes used to practise investigation and detection skills.

## What goes here
- Lab setup notes (VMs, tools, datasets)
- Practice scenarios (benign + malicious behaviour)
- Captured logs and observations (sanitised)
- Lessons learned / fixes

  ## Naming convention
  Use: 'YYYY-MM-DD_LAB-topic.md'

  ## Minimum for each lab note
  - Objective
  - Setup
  - Steps taken
  - What you observed
  - What you'd improve next time
