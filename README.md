# CyberPortfolio
Practical cybersecurity learning portfolio: SOC triage, detections, incident write-ups, and labs.

## Focus 
- Alert triage and investigations
- Detection engineering (Splunk SPL / Sentinel KQL)
- Incident reporting (clear, manager-friendly)
- Windows/Linux log basics and network fundamentals

## Structure
- 'writeups/' : investigation write-ups and lesson learned
- 'detections/' : detection ideas and queris + tuning notes
- 'playbooks/' : step-by-step triage playbooks
- 'labs/' : hands-on lab notes and scenarios
- 'scripts/' : small helper scripts for investigations
